# movie-recommendation-system
A Movie Recommendation System suggests movies to users based on their preferences or similarity between movies. It analyzes features like genre, cast, and keywords using machine learning techniques such as cosine similarity. Platforms like Netflix and Amazon use such systems to provide personalized movie suggestions

Technologies Used
Python
Pandas
NumPy
Scikit-learn
Jupyter Notebook

Algorithm Used
This project uses Content-Based Filtering.
Steps:
Data preprocessing and cleaning
Combining important movie features
Converting text data into vectors
Calculating similarity using cosine similarity
Recommending movies based on similarity scores

Dataset
link- https://www.kaggle.com/datasets/tmdb/tmdb-movie-metadata
The dataset contains information about movies such as:
Movie title
Genre
Cast
Keywords
Overview

How to Run the Project
Clone the repository
Install required libraries
Open the Jupyter Notebook
Run all cells to generate movie recommendations
